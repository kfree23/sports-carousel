# Sports Card Carousel

A click-through carousel showcasing sports cards from multiple sports. Built with vanilla JavaScript, HTML, and CSS.

## Features
- Browse cards with next/previous navigation
- Clean, responsive card display
- Smooth transitions between cards
- Hover animation to show the back

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
